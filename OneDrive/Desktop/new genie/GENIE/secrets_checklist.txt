
# GENIE Infrastructure Secrets Checklist (GitHub Actions)
Add these under: GitHub Repo → Settings → Secrets → Actions

1. TF_API_TOKEN = <Terraform Cloud Token>
2. EC2_SSH_KEY = <Private key from EC2 .pem>
3. SLACK_WEBHOOK = <Slack webhook URL>
4. COHERE_API_KEY = <Cohere.com API key>
5. CLAUDE_API_KEY = <Bedrock access or Claude proxy key>
6. PREDICTNOW_API_KEY = <PredictNow API Key>
